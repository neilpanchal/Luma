Luma

A library by Neil Panchal for the Processing programming environment.
Last update, 04/08/2017.

Luma is a palette generator library for Processing inspired by 'iwanthue' developed by Mathieu Jacomy.
Luma utilizes perceptually linear colorspaces CIE-Lab & CIE-LCH to provide a coherent, equidistant and percetually pleasing color palette.

Download

Download Luma version 1.1.1 (1) in .zip format.

Installation

Unzip and put the extracted Luma folder into the libraries folder of your Processing sketches. Reference and examples are included in the Luma folder.
